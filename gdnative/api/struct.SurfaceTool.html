pub fn generate_normals(&self, flip: bool)

Generates normals from vertices so you do not have to do it manually. If flip is true, the resulting normals will be inverted. generate_normals should be called after generating geometry and before committing the mesh using commit or commit_to_arrays. Note: generate_normals only works if the primitive type to be set to Mesh.PRIMITIVE_TRIANGLES.

Default Arguments

  • flip - false

pub fn generate_tangents(&self)

pub fn index(&self)
